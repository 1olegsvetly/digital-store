# Digital Store — Магазин цифровых товаров

Flat-file CMS для продажи электронных товаров на PHP + JSON.

## 🚀 Возможности

- **Без базы данных** — все данные хранятся в JSON-файлах
- **SEO-оптимизация** — SSR, микроразметка Schema.org, sitemap.xml
- **Админ-панель** — управление товарами, статьями, заказами
- **Импорт CSV** — массовая загрузка товаров и статей
- **Кэширование** — файловый кэш для ускорения отдачи страниц
- **Адаптивный дизайн** — работает на всех устройствах

## 📁 Структура проекта

```
/data           # JSON-файлы с данными (защищено)
  /products     # Товары по категориям
  /articles     # Статьи блога
  /orders       # Заказы
  /users        # Пользователи (админы)
  /config       # Настройки сайта
/admin          # Админ-панель
/catalog        # Каталог товаров
/blog           # Блог
/css            # Стили
/js             # JavaScript
/includes       # PHP-функции
/templates      # Шаблоны страниц
```

## 🔐 Доступ в админку

- **URL:** `/admin/`
- **Логин:** `admin`
- **Пароль:** `admin`

> ⚠️ Сразу после установки смените пароль!

## 🛠 Установка

1. Загрузите файлы на сервер с PHP 8.0+
2. Убедитесь, что папки `data/`, `cache/`, `sessions/` доступны для записи
3. Откройте сайт в браузере
4. Войдите в админку и настройте сайт

## 📝 Импорт товаров из CSV

Формат CSV для импорта:
```csv
title,description_short,description_full,price,currency,stock,min_order,guarantee_hours,category,slug
"Аккаунты FB","Краткое описание","Полное описание",0.148,USD,1000,100,48,facebook,"facebook-accounts"
```

## 🔧 Настройка

Отредактируйте `/data/config/site.json`:
- `site_name` — название сайта
- `site_url` — URL сайта (для sitemap)
- `email`, `phone` — контакты
